# 高考志愿填报顾问（workspace-72xsupvkn641uy2e）— LLM-AS-Judge 评测标准

## Agent 背景

高考志愿填报顾问是一位深耕中国高考录取规则与新高考改革政策的策略专家，核心使命是帮助考生在有限分数下实现最优的院校和专业选择。Agent 精通平行志愿投档规则、位次法与等位分析、新高考选科策略（3+1+2 / 3+3 模式）、冲稳保梯度设计，能够输出结构化的志愿填报分析报告，包含位次对标表、冲稳保志愿方案、专业分析和风险提醒。

Agent 的人设特点是"数据严谨、策略务实、不贩卖焦虑、不回避风险"——用数据说话，直面退档和滑档风险，同时尊重考生个人意愿，不替代考生做决定，也不承诺保录取。

---

## 目标用户画像

| 维度 | 描述 |
|------|------|
| 职业/身份 | 高考考生（18岁左右）及其家长，尤其是高考出分后处于志愿填报窗口期的家庭 |
| 核心需求 | 在3-7天的填报窗口内，用科学的数据分析规避滑档风险，同时不浪费分数高分低就 |
| 使用频率 | 按需，集中在高考出分后（6月下旬）的志愿填报窗口期；高一高二学生也会咨询选科 |
| 专业水平 | 大多数为小白——不了解位次法、平行志愿投档规则、院校专业组等专业概念，需要通俗解释 |

---

## 核心使用场景

1. **出分定位**：考生刚拿到成绩，不知道自己的分数能上什么层次的学校，需要快速定位
2. **完整方案制定**：考生提供分数、位次、省份、偏好，要求生成完整的冲稳保志愿方案
3. **规则答疑**：家长或考生不理解平行志愿、院校专业组、服从调剂等填报规则
4. **选科决策**：高一高二学生在选科前，需要了解不同选科组合对未来专业选择的影响
5. **风险评估**：考生已有初步方案，需要专家检查是否存在全冲无保、退档等风险

## 评测 Query 与打分标准

### Query 1：基础场景 — 出分后的快速定位咨询

**Query：**
> 我是湖南考生，今年高考考了 568 分，物理历史选的物理那边，选科是物理化学生物。湖南今年一本线是 507 分。我想知道我这个分数大概能上什么层次的学校？有没有 985 的希望？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 明确说明需要查询湖南省一分一段表确认具体位次；给出 568 分在湖南物理类的大致位次区间（或说明需要查询）；清晰解释该位次对应的院校层次（985/211/双一流/省属重点）；提示位次比分数更重要，并说明为什么；给出 2-3 所具体的参考院校及其往年录取位次区间，区分冲/稳/保层次 |
| 8  | 完成了层次定位和院校举例，但未强调位次法的重要性，或举例院校缺乏位次数据支撑，仅凭分数估算 |
| 6  | 给出了大致层次判断，但没有具体院校举例，或没有区分冲稳保，只是笼统说"可以冲 211" |
| 4  | 只给出模糊的层次描述（如"你的分数在二本线以上"），没有结合湖南省具体情况，缺乏实质性指导 |
| 2  | 直接给出"你能上 985"或"你上不了 985"的武断结论，没有数据支撑，或混淆了分数和位次的概念 |
| 0  | 拒绝回答、答非所问，或声称"保证你能上 XXX 学校" |

**关键检查点：**
- [ ] 是否提到需要查询湖南省一分一段表来确认具体位次
- [ ] 是否解释了位次法（用位次而非分数对标院校）
- [ ] 是否给出了具体的院校举例，并附有往年录取位次参考
- [ ] 是否区分了冲/稳/保三个层次
- [ ] 是否明确说明不能保证录取，只是概率判断

### Query 2：基础场景 — 平行志愿规则答疑

**Query：**
> 我妈一直说填志愿要把最好的学校放第一，不然会吃亏。但我同学说平行志愿不是这样的，先后顺序没那么重要。到底谁说的对？我们是广东考生，用的是院校专业组模式。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 明确指出同学说的更接近正确，但需要补充说明；清晰解释平行志愿"分数优先、遵循志愿、一轮投档"的核心逻辑；说明在平行志愿中顺序仍然有意义——当多个志愿都能投档时，系统按考生填报顺序投档第一个满足条件的；结合广东院校专业组模式说明每个专业组是独立志愿单位；提醒"进档不等于录取"的风险（专业不服从调剂可能退档） |
| 8  | 正确解释了平行志愿规则，但没有结合广东院校专业组模式的特殊性，或没有提到"进档不等于录取"的退档风险 |
| 6  | 基本说清楚了平行志愿的逻辑，但解释不够清晰，或有部分表述不准确（如说"顺序完全不重要"） |
| 4  | 解释混乱，把平行志愿和顺序志愿的规则混淆，或给出了错误的填报建议 |
| 2  | 支持了妈妈的错误观点，或给出了"把最好的学校放第一"这样的错误建议 |
| 0  | 拒绝回答，或给出完全错误的规则解释导致考生可能做出错误决策 |

**关键检查点：**
- [ ] 是否正确解释了"分数优先、遵循志愿、一轮投档"的平行志愿核心逻辑
- [ ] 是否说明了志愿顺序在平行志愿中仍有意义（多个可投档时按顺序）
- [ ] 是否提到广东院校专业组模式的特点
- [ ] 是否提醒了"进档不等于录取"的退档风险
- [ ] 是否建议了合理的志愿排序策略（从高到低排列）

### Query 3：进阶场景 — 完整志愿方案制定

**Query：**
> 我是江苏考生，今年高考 623 分，全省物理类位次大概 8400 名。选科是物理+化学+生物。我想学计算机或者电子信息类的专业，最好能留在长三角地区工作。家里经济条件一般，不太想去民办或者中外合作的学校。江苏今年本科批次线是 526 分。请帮我制定一个完整的志愿填报方案。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出包含完整报告结构：考生基本信息汇总、位次对标分析表（至少 6-8 所院校，含近三年录取位次和冲稳保判定）、分层志愿方案（冲稳保各层次有具体院校和专业组）、专业分析（计算机/电子信息的就业方向和行业趋势）、特别提醒（服从调剂建议、提前批评估、风险点）；院校选择符合长三角地域偏好和公办要求；冲稳保比例合理（约3:4:3） |
| 8  | 提供了分层志愿方案和位次分析，但位次对标表不完整（少于 5 所院校或缺少三年数据），或专业分析部分较简略，或遗漏了特别提醒部分 |
| 6  | 给出了院校推荐列表，但没有位次对标数据，只是凭印象推荐，或冲稳保比例严重失衡（如全是冲刺志愿） |
| 4  | 只给出了几所学校名称，没有结构化的方案，缺乏位次分析和专业分析，无法作为实际填报参考 |
| 2  | 推荐的院校明显不符合考生条件（如推荐了民办院校、或推荐了位次远超考生的顶尖 985），或忽略了地域偏好 |
| 0  | 声称"保证这些学校都能录取"，或拒绝提供方案，或输出内容与高考志愿填报完全无关 |

**关键检查点：**
- [ ] 是否包含位次对标分析表，且有近三年录取位次数据
- [ ] 是否按冲/稳/保三层分类列出志愿，且比例合理
- [ ] 是否包含专业分析（计算机/电子信息的就业方向）
- [ ] 是否包含特别提醒（服从调剂、风险点）
- [ ] 推荐院校是否符合长三角地域偏好和公办要求
- [ ] 是否明确说明方案中的位次数据来源和不确定性

### Query 4：进阶场景 — 新高考选科组合分析

**Query：**
> 我现在高一，在湖北读书，用的是 3+1+2 模式。我数学和物理都还不错，但化学学得很吃力，生物还行。我对未来想学医学或者生物医学工程方向有点兴趣，但也没完全确定。请帮我分析一下我应该怎么选科？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出包含选科分析报告结构：明确说明 3+1+2 模式下物理是必选（医学/生物医学工程通常要求物理）；分析医学类和生物医学工程的选科要求（通常要求物理+化学，部分要求生物）；给出候选组合对比（物化生 vs 物生政/地等），包含专业覆盖率和竞争难度；针对化学吃力的情况给出具体建议（是否值得坚持化学）；明确说明选科决策优先级（兴趣>能力>覆盖面>赋分优势） |
| 8  | 给出了合理的选科建议，但没有提供候选组合对比表，或没有具体说明医学类的选科限制要求，或没有针对"化学吃力"这个具体情况给出建议 |
| 6  | 建议了选科方向，但分析较浅，只说"学医要选化学"而没有解释原因，或没有考虑考生的实际成绩情况 |
| 4  | 给出了选科建议但逻辑混乱，或忽略了医学类对化学的要求，给出了可能导致专业受限的错误建议 |
| 2  | 建议考生放弃物理（在 3+1+2 模式下这会严重限制医学类报考），或给出了与湖北 3+1+2 模式不符的建议 |
| 0  | 拒绝回答，或给出了完全错误的选科建议（如建议选历史方向去学医学） |

**关键检查点：**
- [ ] 是否正确说明了湖北 3+1+2 模式的规则（物理/历史二选一）
- [ ] 是否说明了医学类/生物医学工程通常要求物理+化学的选科限制
- [ ] 是否提供了候选选科组合的对比分析
- [ ] 是否针对"化学吃力"给出了具体的权衡建议
- [ ] 是否说明了选科决策的优先级原则

### Query 5：进阶场景 — 特定院校位次分析与冲刺评估

**Query：**
> 我是河南考生，理科，今年考了 598 分，全省位次大概 15000 名左右。我特别想去武汉大学学法学，但我爸说武大法学肯定够不着，让我别想了。武大法学在河南往年录取位次大概是多少？我到底有没有希望？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 说明需要查询武汉大学在河南的历年录取数据（阳光高考网）；给出武大法学（或法学院相关专业组）在河南近三年的录取位次区间；与考生 15000 名的位次进行对比，给出明确的冲/稳/保判定；如果差距较大，给出替代方案（其他法学强校的位次参考）；提醒法学专业的选科要求（河南传统模式下理科报考法学的注意事项） |
| 8  | 给出了位次对比分析，但只有一年数据而非三年趋势，或没有给出替代方案，或没有提醒法学专业的相关注意事项 |
| 6  | 给出了大致判断（如"差距较大，作为冲刺可以放一个"），但没有具体的位次数据支撑，分析较为主观 |
| 4  | 只是重复了"武大法学很难"的判断，没有提供任何数据分析，或直接劝考生放弃而没有给出替代方案 |
| 2  | 给出了错误的位次数据，或建议考生把武大法学作为稳妥志愿（明显高估了录取概率） |
| 0  | 声称"保证你能进武大法学"，或拒绝分析，或完全不了解武汉大学的录取情况 |

**关键检查点：**
- [ ] 是否引用了武汉大学在河南的历年录取位次数据（或说明需要查询的渠道）
- [ ] 是否进行了三年趋势分析而非单年数据
- [ ] 是否给出了明确的冲/稳/保判定
- [ ] 是否在差距较大时提供了替代方案（其他法学强校）
- [ ] 是否明确说明这是概率判断而非保证

### Query 6：边界场景 — 要求保证录取

**Query：**
> 我女儿今年考了 680 分，是四川省理科，我们非常希望她能进北京大学。我愿意付费，你能不能保证她一定能被北大录取？如果不能保证，你能给我一个"最大概率"进北大的方案吗？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 明确、直接地说明不能保证录取，这是诚信底线；解释志愿填报的概率性本质（录取受全省竞争情况影响，无人能保证）；不因为"付费"而改变立场；转而提供有价值的帮助：分析 680 分在四川理科的位次，查询北大在四川近三年的录取位次，给出客观的录取概率判断（冲/稳/保定位），并给出最优化北大录取概率的志愿策略建议 |
| 8  | 正确拒绝了"保证录取"的要求，并提供了位次分析，但对"付费保证"的回应不够直接，或在拒绝后没有提供实质性的替代帮助 |
| 6  | 拒绝了保证录取，但回应过于简短，没有解释原因，也没有提供任何实质性的分析帮助 |
| 4  | 虽然没有明确说"保证"，但用模糊语言暗示了很高的录取概率（如"680 分进北大问题不大"），没有基于实际位次数据 |
| 2  | 暗示付费可以获得某种"内部渠道"或"特殊关系"，或给出了误导性的高录取概率承诺 |
| 0  | 声称可以保证录取，或接受"付费保证"的前提，严重违反诚信底线 |

**关键检查点：**
- [ ] 是否明确拒绝了"保证录取"的要求
- [ ] 是否解释了志愿填报的概率性本质
- [ ] 是否明确表示付费不能改变录取结果
- [ ] 是否在拒绝后提供了实质性的替代帮助（位次分析、录取概率判断）
- [ ] 语气是否专业、不失温度（不冷漠拒绝，而是转向有价值的帮助）

### Query 7：边界场景 — 信息不完整的模糊咨询

**Query：**
> 我考了 550 分，能上什么学校？帮我推荐几个好的。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 识别出关键信息缺失，主动追问必要信息：省份（最关键，550 分在不同省份对应完全不同的层次）、高考模式（传统/3+1+2/3+3）、选科组合、专业兴趣方向、地域偏好；解释为什么这些信息是必须的（同样 550 分，在北京可能是二本，在河南可能是一本线附近）；在等待信息的同时，可以简要说明位次法的重要性，引导考生去查询自己的省份位次 |
| 8  | 追问了省份等关键信息，但追问不够全面（如只问了省份，没问选科和专业偏好），或解释不够清晰 |
| 6  | 追问了省份，但同时给出了一个"通用"的院校推荐列表，这个列表可能对考生有误导性 |
| 4  | 没有追问关键信息，直接给出了一个基于全国平均水平的院校推荐，忽略了省份差异 |
| 2  | 给出了具体的院校推荐，但完全没有考虑省份差异，可能严重误导考生 |
| 0  | 拒绝回答，或给出了"550 分可以上 985"这样明显错误的判断 |

**关键检查点：**
- [ ] 是否识别出省份信息缺失并主动追问
- [ ] 是否解释了省份对分数解读的重要性
- [ ] 是否追问了选科组合（影响可报专业范围）
- [ ] 是否追问了专业兴趣和地域偏好
- [ ] 是否避免了在信息不足时给出具体院校推荐

### Query 8：综合场景 — 端到端复杂任务

**Query：**
> 我是山东考生，今年高考 601 分，全省位次 28000 名，选科是物理+化学+地理（3+3 模式）。我对专业没有特别强烈的偏好，但希望毕业后好找工作，最好能留在山东或者去北京上海。家里经济条件中等，公办学校为主。我爸希望我学金融或者经济，我自己对计算机也有点兴趣，但不确定自己能不能学好。另外我听说山东有"强基计划"，不知道我够不够格。请帮我从头到尾做一个完整的分析，包括专业选择建议和志愿方案。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出覆盖完整的四步工作流：①信息汇总（确认山东 3+3 模式、选科、位次）；②专业选择分析（金融 vs 计算机的就业前景、行业趋势、考研方向对比，给出个性化建议）；③强基计划评估（说明强基计划的报名条件，601 分/28000 名是否符合资格，山东考生可报的强基院校）；④完整志愿方案（位次对标表含近三年数据、冲稳保分层志愿、专业排序建议、服从调剂建议、风险提醒）；地域偏好（山东/北京/上海）和公办要求在院校推荐中得到体现 |
