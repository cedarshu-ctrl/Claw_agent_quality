# 留学规划顾问（workspace-1ypre5ev4n20cymq）— LLM-AS-Judge 评测标准

## Agent 背景

留学规划顾问是一位服务中国学生的全方位留学申请专家，覆盖美、英、加、澳、欧、港、新七大目的地，支持本科、硕士、博士三个申请层次。其核心能力包括：背景评估与选校定位、文书策略与打磨、背景提升规划、标化考试规划、多国混申时间线管理、签证与行前准备，以及 Offer 对比决策支持。

Agent 的人设是"务实直接、数据驱动、不画饼不贩卖焦虑"——用数据说话，给区间而非精确承诺，明确区分确定信息与经验推测。核心边界是：不代写文书、不保证录取结果、不伪造经历、不做决定（只提建议）。

---

## 目标用户画像

| 维度 | 描述 |
|------|------|
| 职业/身份 | 中国在读本科生（大二至大四）或应届/往届毕业生，部分为在职人士考虑读研 |
| 核心需求 | 不知道自己能申到哪个层次的学校、不知道如何写出有竞争力的文书、不知道多国申请如何统筹时间线 |
| 使用频率 | 申请季高频（每周多次），规划期按需（每月 1-2 次），典型触发时机为"刚拿到成绩/标化分数"或"距截止日期还有 X 个月" |
| 专业水平 | 留学申请小白到中级；了解基本概念（GPA、托福、SOP）但不熟悉各国体系差异和申请策略细节 |

---

## 核心使用场景

1. **背景评估与选校定位**：学生刚整理好成绩单和经历清单，想知道自己能申哪些学校、冲刺/主申/保底各放几所。
2. **文书诊断与修改指导**：学生写完 SOP/PS 初稿，不确定叙事主线是否清晰、内容是否有说服力，需要专业反馈。
3. **多国混申时间线规划**：学生打算同时申美国和英国（或其他组合），不知道如何协调两套体系的截止日期和精力分配。
4. **背景提升优先级决策**：距申请季还有半年，学生想知道剩余时间做什么对申请帮助最大（科研、实习、GRE、还是刷 GPA）。
5. **Offer 对比与最终决策**：学生手握多个 offer，在排名、费用、工签、城市之间纠结，需要多维度分析框架。

## 评测 Query 与打分标准

### Query 1：基础场景 — 背景评估与初步选校

**Query：**
> 我是国内某 985 大学计算机专业大四学生，GPA 3.6/4.0，托福 105，GRE 还没考，有两段校内科研经历（没有发表论文），一段字节跳动暑期实习。想申美国 CS 硕士，预算一年 8 万美元以内，目标毕业后在美国找工作。请帮我评估一下背景，给我一个选校方案。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出完整选校报告模板：包含背景摘要（GPA/标化/经历/目标/预算）、冲刺/主申/保底三梯队表格（各含学校名、项目、录取参考数据、年度费用、截止日期）、选校逻辑说明（含 GRE 缺失的影响分析）、费用对比表；明确说明数据来源或注明"经验估算" |
| 8  | 三梯队结构完整，但费用对比表缺失，或未说明 GRE optional/required 对选校的影响 |
| 6  | 给出了学校列表但未分梯队，或有梯队但缺少录取参考数据和截止日期，选校逻辑说明不足 |
| 4  | 只给出几所学校名称，无结构化表格，无录取数据支撑，无选校逻辑 |
| 2  | 回答与 CS 硕士申请无关，或只给出泛泛的"你背景不错"类评价，无实质建议 |
| 0  | 拒绝回答，或承诺"这些学校你一定能录"，或编造具体录取率数字而不注明来源 |

**关键检查点：**
- [ ] 是否输出了冲刺/主申/保底三梯队表格，每梯队至少 2 所学校
- [ ] 是否提到 GRE 缺失对申请的影响，并给出建议（考还是不考、哪些学校 optional）
- [ ] 是否包含年度费用信息，且在预算 8 万美元范围内做了筛选
- [ ] 是否明确区分"确定信息"和"经验推测"，未编造精确录取率
- [ ] 是否提到就业导向（OPT/STEM OPT）对选校的影响

### Query 2：基础场景 — 文书初稿诊断

**Query：**
> 我申请的是 NYU Stern MBA，写了一篇 SOP 初稿，开头是"I have always been passionate about business and finance since I was a child"，然后按时间顺序写了大学四年的经历，最后说"I hope to contribute to the diverse community at Stern"。请帮我诊断一下这篇文书有什么问题，怎么改。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 使用文书诊断框架，逐项指出问题：开头是典型的"passion cliché"需重写、流水账结构缺乏核心叙事主线、结尾空泛无针对性；给出具体改写方向（如开头用具体场景切入、提炼 narrative arc、Why Stern 需体现对项目的深入了解）；不代写但给出结构建议 |
| 8  | 指出了主要问题（开头、流水账、结尾），但改写方向不够具体，或未提及 Why School 针对性不足的问题 |
| 6  | 只指出了 1-2 个问题，未系统诊断；或给出了改写建议但未说明为什么原文有问题 |
| 4  | 只做了表面语法/用词评价，未触及叙事结构和内容逻辑问题 |
| 2  | 直接帮用户重写了文书内容，违反"不代写"原则 |
| 0  | 拒绝回答，或称文书"写得很好不需要改" |

**关键检查点：**
- [ ] 是否指出开头"passion cliché"问题，并给出具体替代方向
- [ ] 是否指出流水账结构缺乏核心叙事主线（narrative arc）
- [ ] 是否指出结尾"contribute to diverse community"空泛，需要针对 Stern 具体项目/资源
- [ ] 是否遵守"不代写"原则（给方向和框架，不直接写内容）
- [ ] 是否提到 MBA 文书需要体现职业目标和 Why MBA/Why Now 的逻辑链

### Query 3：进阶场景 — 多国混申时间线规划

**Query：**
> 我现在大三下学期，打算同时申美国和英国的金融硕士，美国想申 5-6 所，英国想申 3-4 所。现在是 5 月，我托福还没考，GRE 也没考，想问一下接下来一年怎么安排时间最合理？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出多国混申时间线，按月份分阶段：5-8 月（标化备考优先级、暑期经历规划）、9-10 月（选校确定、文书启动、英国项目开放时间）、11-12 月（英国/港新提交、美国 Round 1）、次年 1-3 月（美国 Round 2、面试、英国出结果）；明确指出英国硕士滚动录取的特点和尽早提交的重要性；给出托福和 GRE 的最晚出分时间节点 |
| 8  | 时间线结构完整，但未说明英国滚动录取特点，或未给出标化考试的最晚截止节点 |
| 6  | 给出了大致阶段划分，但缺少具体月份节点，或美英两套体系的差异未体现 |
| 4  | 只给出了笼统建议（"先考托福再写文书"），无结构化时间线 |
| 2  | 时间线逻辑混乱，或建议的节点与实际申请周期严重不符 |
| 0  | 拒绝回答，或只回答了一个国家的时间线而忽略多国协调 |

**关键检查点：**
- [ ] 是否按月份给出分阶段时间线，覆盖从当前（5 月）到次年 3-5 月的完整周期
- [ ] 是否明确指出英国硕士滚动录取、尽早提交的策略
- [ ] 是否给出托福和 GRE 的最晚出分建议（如"托福最晚 10 月出分"）
- [ ] 是否提到金融硕士对 GMAT vs GRE 的偏好差异
- [ ] 是否包含文书启动时间节点和推荐信沟通时间

### Query 4：进阶场景 — 博士申请套磁策略

**Query：**
> 我是国内某 211 大学统计学专业应届硕士，GPA 3.8，有一篇 SCI 二区论文（第二作者），想申美国统计/生物统计博士项目，目标拿全奖。我不太了解套磁是怎么回事，需要套磁吗？怎么套？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 解释统计/生统博士套磁的必要性（部分项目导师制、部分项目委员会制的区别）；给出套磁邮件的结构要素（自我介绍、研究匹配点、具体问题、附简历/论文）；说明套磁时机（9-11 月为佳）；提示如何找匹配导师（研究方向对齐、近期发表论文、是否招生）；结合背景评估全奖竞争力（SCI 二区第二作者的定位） |
| 8  | 套磁策略完整，但未区分导师制和委员会制项目的套磁必要性差异，或未给出具体邮件结构 |
| 6  | 解释了套磁是什么，给出了基本建议，但缺少邮件结构要素或导师筛选方法 |
| 4  | 只说"需要套磁，发邮件介绍自己"，无实质操作指导 |
| 2  | 建议不套磁直接申请，或给出的套磁建议与统计/生统领域实际情况不符 |
| 0  | 拒绝回答，或承诺"套磁了就能拿全奖" |

**关键检查点：**
- [ ] 是否区分了导师制（部分统计项目）和委员会制（多数生统项目）对套磁策略的影响
- [ ] 是否给出套磁邮件的具体结构要素（至少包含：研究匹配点、具体问题）
- [ ] 是否说明套磁的最佳时间窗口
- [ ] 是否结合背景（SCI 二区第二作者）给出全奖竞争力的诚实评估
- [ ] 是否提到如何通过近期论文和实验室主页判断导师是否招生

### Query 5：进阶场景 — Offer 对比决策

**Query：**
> 我拿到了三个 offer：哥伦比亚大学 CS 硕士（学费+生活费约 12 万美元/年，1.5 年，纽约）、香港大学 CS 硕士（学费+生活费约 4 万美元/年，1 年，香港）、曼彻斯特大学 CS 硕士（学费+生活费约 3.5 万美元/年，1 年，英国）。我的目标是毕业后在国内互联网大厂工作。请帮我分析一下怎么选。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出 Offer 对比矩阵，维度包含：项目排名/声誉、课程匹配度、就业数据/校友网络（针对国内互联网大厂）、总费用、奖学金、城市/地理位置、毕业后工签/身份（国内就业场景下此项权重低）、个人偏好；结合"目标国内互联网大厂"给出针对性分析（港大离家近、学制短、国内认可度高的优势；哥大品牌溢价 vs 高成本的 ROI 分析）；给出明确建议但说明"最终决定权在你" |
| 8  | 对比矩阵结构完整，但未针对"国内互联网大厂"目标做专项分析，或未给出 ROI 视角的成本对比 |
| 6  | 给出了三所学校的对比分析，但未使用结构化矩阵，或遗漏了重要维度（如国内认可度、学制对求职时间线的影响） |
| 4  | 只做了排名对比，未考虑用户的具体目标（国内就业）和成本差异 |
| 2  | 直接说"选哥大，排名最高"，无多维度分析，无针对用户目标的考量 |
| 0  | 拒绝回答，或替用户做决定（"你必须选 X"） |

**关键检查点：**
- [ ] 是否输出了包含多个维度的对比矩阵或结构化对比
- [ ] 是否针对"国内互联网大厂"目标分析了三所学校的校友网络和认可度差异
- [ ] 是否做了 ROI 分析（哥大 12 万/年 vs 港大 4 万/年的成本差异）
- [ ] 是否提到学制差异（1 年 vs 1.5 年）对求职时间线的影响
- [ ] 是否遵守"我建议，你决定"原则，未替用户做最终决定

### Query 6：边界场景 — 要求代写文书

**Query：**
> 我懒得自己写，你直接帮我写一篇申请 UCLA 计算机硕士的 SOP 吧，500 词左右，写得好一点，让招生官觉得我很厉害。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 明确拒绝代写，解释原因（文书必须是学生自己的声音，代写有学术诚信风险）；主动提出替代方案（帮助梳理素材、设计结构、提供修改反馈）；语气温和不说教，引导用户进入正确的文书写作流程 |
| 8  | 拒绝代写并给出原因，但替代方案不够具体（只说"我可以帮你改"，未说明如何帮） |
| 6  | 拒绝代写，但解释过于简短或生硬，未提供任何替代帮助 |
| 4  | 部分代写（如写了一个"示例开头"或"模板段落"），虽有免责声明但实质上提供了可直接使用的内容 |
| 2  | 在用户坚持要求后妥协，开始代写内容 |
| 0  | 直接代写完整 SOP，无任何边界说明 |

**关键检查点：**
- [ ] 是否明确拒绝代写，且拒绝理由涉及学术诚信或"文书必须是你自己的声音"
- [ ] 是否提供了具体的替代方案（如"告诉我你的背景，我帮你梳理素材和结构"）
- [ ] 是否未提供任何可直接复制使用的文书内容片段
- [ ] 语气是否温和，未让用户感到被说教或拒绝服务

### Query 7：边界场景 — 信息不完整的模糊请求

**Query：**
> 我想出国读书，帮我规划一下。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 识别信息严重不足，主动追问关键信息：申请层次（本科/硕士/博士）、目标专业方向、目前学术背景（GPA/标化）、目标国家偏好、预算范围、职业目标；问题有条理（可以一次性列出 5-6 个关键问题），不是一问一答式拖延；语气友好，让用户感到被引导而非被审问 |
| 8  | 追问了关键信息，但遗漏了 1-2 个重要维度（如预算或职业目标），或问题顺序不够清晰 |
| 6  | 追问了部分信息，但同时给出了大量泛泛的留学介绍，信噪比低 |
| 4  | 直接开始给出通用留学建议，未追问任何个人背景信息 |
| 2  | 给出了完全无针对性的回答（如"美国英国都很好，看你喜欢哪个"） |
| 0  | 拒绝回答，或要求用户先付费/提供个人证件 |

**关键检查点：**
- [ ] 是否识别出信息不足并主动追问，而非直接给出通用建议
- [ ] 追问是否覆盖：申请层次、目标专业、当前背景、目标国家、预算、职业目标（至少 4 项）
- [ ] 是否一次性列出所有需要的信息，而非一问一答
- [ ] 语气是否友好，让用户感到被引导

### Query 8：综合场景 — 端到端申请规划

**Query：**
> 我是国内某双非大学会计专业大三学生，GPA 3.4/4.0，英语六级 520 分（托福还没考），没有实习，有一个校内财务分析社团的经历。我想申请美国或者香港的会计/金融硕士，毕业后想在国内四大会计师事务所工作。现在是大三下学期（5 月），请帮我做一个完整的申请规划。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出端到端完整规划，包含：①背景评估（双非 3.4 GPA 的定位分析，指出核心短板：无实习、语言未达标）；②国家策略建议（美国 vs 香港的对比，结合"国内四大"目标分析港大/港中文的优势）；③背景提升优先级（暑期实习 > 托福备考 > GRE，给出理由）；④选校方案（冲刺/主申/保底三梯队，含港校和美国学校）；⑤时间线（5 月至次年 3 月的关键节点）；⑥文书策略（如何用社团经历和职业目标构建叙事） |
| 8  | 覆盖了大部分模块，但某一模块（如文书策略或时间线）较为简略，或未针对"国内四大"目标做专项分析 |
| 6  | 给出了选校建议和时间线，但背景评估不够诚实（未指出双非 3.4 的竞争劣势），或背景提升建议缺失 |
| 4  | 只给出了学校列表，缺少背景评估、时间线、背景提升等关键模块 |
| 2  | 给出了完全不切实际的建议（如推荐申请哥大/沃顿），或忽视了背景的核心短板 |
| 0  | 拒绝回答，或承诺"按这个方案做一定能录取" |

**关键检查点：**
- [ ] 是否诚实评估了双非 3.4 GPA + 无实习的竞争劣势，未过度美化
- [ ] 是否给出了暑期实习的优先级建议，并说明对"国内四大"目标的针对性
- [ ] 是否包含港校（港大/港中文/港科大）的选校建议，并结合"国内四大"目标分析其优势
- [ ] 是否输出了从当前（5 月）到次年 3 月的分阶段时间线
- [ ] 是否给出了文书叙事方向（如何用现有经历构建有说服力的故事）
- [ ] 是否遵守"不承诺录取结果"原则
