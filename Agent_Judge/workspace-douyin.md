# 抖音策略师（workspace-58eu30pmge0d03m1）— LLM-AS-Judge 评测标准

## Agent 背景

抖音策略师是一位深谙抖音推荐算法逻辑的短视频营销专家，核心定位是"让视频上热门不靠玄学"。其能力覆盖短视频内容策划（黄金3秒开头设计、完播率优化、脚本结构）、流量运营与投放（DOU+、千川、矩阵账号）、直播带货（排品策略、话术设计、数据复盘）四大模块，并遵循严格的合规红线（不使用绝对化用语、广告法合规）。

该 Agent 的标志性交付物包括：**爆款视频脚本**（含黄金3秒开头+核心内容+结尾钩子的完整结构）和**直播排品表**（含商品结构比例+2小时直播节奏表）。沟通风格直接高效、数据驱动，拒绝废话，优先给出可立即执行的行动建议。

## 核心使用场景

1. **新账号冷启动**：刚注册账号，不知道从哪里开始，需要完整的定位和内容规划方案
2. **单条视频优化**：拍了视频但播放量卡在几百，想知道哪里出了问题、怎么改
3. **直播间起号**：想开直播带货但不知道怎么排品、怎么控节奏、怎么留住观众
4. **DOU+ 投放决策**：想花钱推广但怕打水漂，需要知道怎么选人群、投多少合适
5. **爆款内容复制**：看到同类账号某条视频爆了，想拆解结构并复制到自己账号

## 评测 Query 与打分标准

### Query 1：基础场景 — 新账号第一条视频脚本

**Query：**
> 我刚开了一个抖音账号，准备卖自家做的手工辣酱，第一条视频不知道怎么拍。我没有拍视频的经验，就是个普通人，能帮我写一个完整的视频脚本吗？产品是四川风味的辣酱，主打纯手工、无添加，定价39元一瓶。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出完整脚本，包含：①明确标注时长目标（30-45秒）；②黄金3秒开头（从冲突型/利益型/悬念型/共鸣型中选一种并写出具体文案）；③第4-20秒核心内容（痛点放大→解决方案→演示→数据/对比）；④结尾钩子（引导互动+系列预告）；⑤拍摄要求（竖屏9:16、字幕、BGM建议）；⑥给出完播率优化说明 |
| 8  | 脚本结构完整，但缺少拍摄要求说明，或开头类型未明确标注，或结尾钩子只有互动引导没有系列预告 |
| 6  | 有脚本内容但结构不完整，缺少黄金3秒的具体文案（只说"要有吸引力的开头"），或缺少完播率优化策略说明 |
| 4  | 给出了一些内容建议但没有按脚本格式输出，或只给出开头文案没有完整脚本结构 |
| 2  | 输出的是通用的"短视频拍摄技巧"而非针对该产品的具体脚本，或脚本与辣酱产品无关 |
| 0  | 拒绝写脚本，或输出内容完全答非所问，或使用了"最好吃""100%纯手工"等绝对化违规用语 |

**关键检查点：**
- [ ] 黄金3秒开头是否给出了具体可用的文案（而非只说"要有冲突感"）
- [ ] 脚本是否包含第4-20秒的分段内容（痛点→方案→演示→数据）
- [ ] 是否有结尾互动引导语（具体文案，不是泛泛的"引导互动"）
- [ ] 是否提到竖屏9:16和字幕要求
- [ ] 是否避免了"最好""第一""100%"等绝对化用语

### Query 2：基础场景 — 视频播放量低的诊断

**Query：**
> 我发了20条视频了，每条播放量都在300-500左右，涨粉也很慢，就涨了80个粉。我做的是职场穿搭类内容，每条视频大概1分钟左右，都是我自己出镜介绍当天穿搭。数据这么差是哪里出了问题？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 给出系统诊断，包含：①指出1分钟时长对穿搭类内容偏长（建议30-45秒）；②分析完播率可能是核心问题（算法权重排序：完播率>点赞率>评论率>转发率）；③给出前3秒开头的具体改进方向（穿搭类适合哪种开头类型）；④给出至少2条可立即执行的改进动作（如缩短时长、换开头方式、加字幕等）；⑤提到发布时间优化或评论区运营 |
| 8  | 诊断方向正确，给出了具体改进建议，但未提及算法权重逻辑，或改进建议少于2条 |
| 6  | 指出了问题（如时长太长、开头不吸引人），但只给出方向性建议，没有具体可执行的操作步骤 |
| 4  | 给出了一些通用的"做好内容"建议，但没有针对穿搭类账号的具体分析，或没有提到完播率这个核心指标 |
| 2  | 回答偏向安慰性内容（"坚持下去就会好的"），或给出的建议与抖音算法逻辑无关 |
| 0  | 拒绝回答，或建议用户去买粉/刷数据等违规操作 |

**关键检查点：**
- [ ] 是否明确提到完播率是核心问题
- [ ] 是否给出了视频时长的具体建议（穿搭类30-45秒）
- [ ] 是否给出了前3秒开头的具体改进方向
- [ ] 是否包含至少2条可立即执行的具体动作
- [ ] 是否提到算法权重排序逻辑

### Query 3：进阶场景 — 完整直播排品方案

**Query：**
> 我是卖护肤品的，下周六要做一场2小时的直播，主要产品有：9.9元的洗脸巾（引流用）、198元的精华液（主推）、398元的套装（利润款）、699元的高端面霜（形象款）。帮我设计一个完整的直播排品和节奏安排，我想让直播间的人气和销售额都能好看一点。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出完整直播排品表，包含：①商品结构分类（引流款/利润款/形象款/福利款）及各类占比；②2小时直播节奏表（至少6个时间段，含时间、环节名称、对应商品、话术重点）；③每个时间段的核心目标说明（如"拉停留""做互动数据""逼单转化"）；④护肤品类目的合规提示（不过度承诺效果）；⑤GPM等核心数据指标说明 |
| 8  | 排品表结构完整，时间节奏合理，但缺少护肤品合规提示，或话术重点描述过于简略 |
| 6  | 给出了商品排序建议，但没有按时间段输出完整节奏表，或缺少商品结构比例分析 |
| 4  | 只给出了"先卖便宜的再卖贵的"等笼统建议，没有具体的时间节奏和话术方向 |
| 2  | 输出内容与直播排品无关，或建议顺序明显不合理（如一开始就推最贵的形象款） |
| 0  | 拒绝回答，或建议虚假宣传（如"承诺7天美白"等违规话术） |

**关键检查点：**
- [ ] 是否输出了包含时间段的完整节奏表（而非只有文字描述）
- [ ] 是否将9.9元洗脸巾定位为引流/福利款并安排在开场阶段
- [ ] 是否包含"每15分钟一个流量峰值循环"的节奏意识
- [ ] 是否有护肤品类目的合规提示（不过度承诺效果）
- [ ] 是否提到GPM或停留时长等直播核心指标

### Query 4：进阶场景 — DOU+ 投放策略

**Query：**
> 我有一条视频自然播放到了8000，完播率大概35%，内容是教大家怎么用空气炸锅做低卡餐的。我想投DOU+放大这条视频，预算500块，应该怎么投？选什么人群？投多少合适？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 给出完整投放方案，包含：①肯定该视频值得投放的判断依据（完播率35%超过35%基准线）；②人群选择建议（系统智能投放 vs 自定义人群的选择逻辑，针对美食/健康/减脂人群的具体标签）；③投放目标选择（粉丝增长 vs 视频播放的区别和适用场景）；④500元预算的分配建议（如先小额测试再放量）；⑤投放后的数据观察指标（ROI>1:3的判断标准） |
| 8  | 给出了人群选择和预算建议，但缺少投放目标的选择逻辑，或没有提到先小额测试的策略 |
| 6  | 给出了基本的投放建议，但人群标签不够具体，或只说"选精准人群"没有给出具体标签方向 |
| 4  | 只给出了"可以投DOU+"的建议，没有具体的人群、预算分配或投放策略 |
| 2  | 建议直接把500元全部一次性投出，或给出的人群选择与内容方向完全不匹配 |
| 0  | 拒绝回答，或建议刷量等违规操作 |

**关键检查点：**
- [ ] 是否基于35%完播率给出明确的"值得投放"判断
- [ ] 是否给出了具体的人群标签方向（美食/健康/减脂相关）
- [ ] 是否建议先小额测试再放量（而非直接全投）
- [ ] 是否提到DOU+ ROI>1:3的成功指标
- [ ] 是否区分了"粉丝增长"和"视频播放"两种投放目标的适用场景

### Query 5：进阶场景 — 账号定位与内容矩阵规划

**Query：**
> 我是一个健身教练，想在抖音做账号，但不知道该怎么定位。我有10年健身经验，擅长体态矫正和减脂训练，目前在一家健身房上班，想通过抖音接私教客户。帮我规划一下账号定位和未来一个月的内容方向。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出完整规划，包含：①账号定位（人设+内容方向+变现路径三要素）；②竞品分析建议（对标账号类型和分析维度）；③内容矩阵规划（知识类/剧情类/测评类/vlog类中选2-3种适合健身教练的类型，并说明各类型的完播率特点）；④一个月内容计划（建议日更或隔日更，给出具体的选题方向和比例）；⑤变现路径说明（如何从内容引导到私教咨询） |
| 8  | 定位清晰，内容矩阵合理，但缺少变现路径的具体设计，或一个月内容计划过于笼统 |
| 6  | 给出了账号定位建议，但内容矩阵只有方向没有具体选题，或没有结合健身教练的变现目标（接私教客户）来规划内容 |
| 4  | 只给出了"做健身内容"的方向，没有具体的人设设计、内容类型规划或变现路径 |
| 2  | 给出的建议与健身教练的实际情况脱节，或内容规划不考虑完播率优化 |
| 0  | 拒绝回答，或建议购买粉丝/刷数据等违规操作 |

**关键检查点：**
- [ ] 是否明确给出人设定位（不只是"健身教练"，而是更细分的标签如"体态矫正专家"）
- [ ] 是否包含至少2种内容类型及其适用场景
- [ ] 是否给出了具体的更新频率建议（日更/隔日更）
- [ ] 是否将变现目标（接私教客户）融入内容规划
- [ ] 是否提到竞品分析的必要性

### Query 6：边界场景 — 超出能力范围的请求（其他平台）

**Query：**
> 我在抖音做得还不错，现在想把内容同步到小红书和视频号，你能帮我制定一个三平台联动的运营策略吗？重点是小红书的图文内容怎么做，感觉和抖音完全不一样。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 诚实说明自己专注于抖音平台，对小红书图文内容策略不是专长；同时给出抖音视频同步到其他平台的基本注意事项（如去水印、内容适配）；对于视频号可以给出与抖音算法的差异说明；建议用户针对小红书寻找专门的平台策略资源 |
| 8  | 明确说明专注抖音，给出了视频号的基本建议，但对小红书部分给出了一些通用建议而非承认边界 |
| 6  | 尝试回答三平台策略，但小红书部分内容明显不专业或与实际不符，且没有说明自己的能力边界 |
| 4  | 给出了三平台的通用建议，内容质量参差不齐，没有体现抖音专家的专业深度 |
| 2  | 对小红书图文策略给出了自信但错误的建议，误导用户 |
| 0  | 完全拒绝回答，或给出的建议完全脱离实际 |

**关键检查点：**
- [ ] 是否明确说明自己专注于抖音平台
- [ ] 是否对抖音内容同步到其他平台给出了实用建议（如去水印）
- [ ] 是否避免了对小红书图文策略的过度自信表述
- [ ] 是否给出了建设性的替代建议（如推荐寻找小红书专家）

- [ ] 替代方案是否具体可用（而非只说"要合规"）

### Query 8：综合场景 — 新品牌从0到1的抖音增长方案

**Query：**
> 我们是一个新成立的国产咖啡品牌，主打"办公室提神、低糖健康"，客单价在30-50元之间，目前没有抖音账号，也没有做过任何短视频。老板给了我3个月时间，要求账号涨到1万粉，并且通过直播和橱窗实现月销售额10万。我是运营负责人，需要一个完整的执行方案，从账号搭建到内容生产到直播带货都要覆盖。

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出端到端完整方案，覆盖四个阶段：①账号诊断与定位（人设设计、内容方向、竞品分析）；②内容规划与生产（内容矩阵类型、视频脚本结构要求、更新频率）；③流量运营（DOU+投放策略、发布时间优化、评论区运营）；④直播带货（排品策略、直播节奏、话术方向）；同时给出3个月的阶段性目标拆解（第1月冷启动、第2月放量、第3月变现），并说明核心成功指标（完播率>35%、月涨粉率>15%、直播GPM>500） |
| 8  | 覆盖了四个核心模块，但3个月阶段性目标拆解不够具体，或缺少核心成功指标的量化说明 |
| 6  | 覆盖了内容策划和直播两个模块，但流量运营（DOU+投放）或账号定位部分缺失，整体方案不完整 |
| 4  | 给出了一些方向性建议，但没有按阶段拆解，也没有具体的执行步骤，无法直接落地 |
| 2  | 只给出了内容创作的通用建议，没有覆盖流量运营和直播带货，与"从0到1完整方案"的需求严重不符 |
| 0  | 拒绝回答，或给出的方案完全脱离抖音平台实际（如建议去做图文内容、建议买粉等） |

**关键检查点：**
- [ ] 是否覆盖了账号定位、内容生产、流量运营、直播带货四个模块
- [ ] 是否给出了3个月的阶段性目标拆解（而非一个笼统的方案）
- [ ] 是否包含视频脚本的结构要求（黄金3秒+核心内容+结尾钩子）
