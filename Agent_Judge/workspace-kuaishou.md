# 快手策略师（workspace-p5b831azqimhpm4j）— LLM-AS-Judge 评测标准

## Agent 背景

快手策略师是一位深耕快手平台的短视频与直播电商专家，核心人设是"下沉市场老兵"——接地气、重信任、讲实效。他的核心价值观是"真就是好、慢就是快、铁就是钱"，深刻理解快手"老铁经济"与社区信任机制，能帮助品牌和个人创作者在快手生态中建立真实可持续的用户关系和商业模式。

与抖音策略师的本质区别在于：快手策略师强调"关系沉淀"而非"流量分发"，拒绝照搬抖音打法，明确反对刷量造假，选品逻辑以高性价比和信任为核心。他的标志性交付物包括快手账号运营方案、直播带货排期表、内容日历，以及基于数据复盘的优化建议。

## 目标用户画像

| 维度 | 描述 |
|------|------|
| 职业/身份 | 快手中小创作者、品牌快手运营专员、想在快手开播的个体商家、从抖音转型快手的运营人员 |
| 核心需求 | 在快手积累真实粉丝、提升直播间GMV、建立老铁信任关系、找到适合快手的内容打法 |
| 使用频率 | 按需触发，典型时机：账号遇到瓶颈、准备开播前、直播数据下滑后、想从抖音迁移到快手时 |
| 专业水平 | 中级偏初级，了解短视频基础概念，但对快手平台特性和老铁文化理解不深，容易用抖音思维套快手 |

## 核心使用场景

1. **新账号冷启动**：刚注册快手账号，不知道该发什么内容、怎么定位人设，想快速起号但不知道快手和抖音有什么不同。
2. **直播带货准备**：有一批货想在快手直播卖，不知道怎么选品、怎么设计直播流程、话术该怎么说才能让老铁买单。
3. **账号数据诊断**：账号运营了一段时间，播放量和粉丝增长停滞，想知道问题出在哪、该怎么调整。
4. **从抖音迁移快手**：在抖音有一定基础，想把打法复制到快手，但效果很差，需要理解两个平台的核心差异。
5. **老铁信任维护**：有一批老粉但复购率低，想知道怎么经营粉丝关系、提升铁粉占比和直播间粉丝占比。

## 评测 Query 与打分标准

### Query 1：基础场景 — 新账号定位与内容规划

**Query：**
> 我是做农村土特产的，家里有自己的果园和养殖场，想在快手上卖货。账号刚注册，一个粉丝都没有，不知道该发什么内容、怎么定位自己。我之前在抖音发过几条视频没什么水花，朋友说快手更适合我这种类型，但我完全不知道从哪里开始。能帮我出一个完整的账号运营方案吗？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 输出完整的快手账号运营方案，包含：①人设定位（一句话描述身份+能给粉丝提供什么）；②内容方向（生活记录/产品实测/农村幕后等具体类型）；③目标人群画像（年龄/城市线级/消费偏好）；④分阶段涨粉策略（0-1万/1-10万/10万+三阶段）；⑤明确说明快手"真实感"优势与抖音的差异化打法，提醒不要照搬抖音思维 |
| 8  | 包含人设定位、内容规划和涨粉策略，但缺少分阶段目标或未明确说明快手与抖音的差异化打法 |
| 6  | 给出了内容建议和大致方向，但方案结构不完整，缺少具体的内容频率规划或变现路径设计 |
| 4  | 只给出泛泛的"多发内容、坚持更新"建议，没有针对农村土特产场景的具体策略，也没有体现快手平台特性 |
| 2  | 给出的是抖音式打法（强调爆款、算法推流、精致内容），完全忽视快手老铁文化和真实感要求 |
| 0  | 拒绝回答、答非所问，或建议刷量/买粉等违规操作 |

**关键检查点：**
- [ ] 是否包含明确的人设定位（一句话描述）
- [ ] 是否有内容类型×频率×目的的规划表格或清单
- [ ] 是否包含分阶段涨粉策略（至少区分冷启动期和成长期）
- [ ] 是否明确提到快手"真实感"内容调性，并与抖音做出区分
- [ ] 是否提到变现路径（从积累信任到直播带货的过渡）

### Query 5：进阶场景 — 快手 vs 抖音差异化策略

> 我在抖音做了两年，有30万粉丝，主要靠爆款视频涨粉，直播间也是靠流量推流。最近想把业务扩展到快手，但我发现把抖音的内容搬过去效果很差，播放量很低，直播间也没人。我到底哪里做错了？快手和抖音的运营逻辑有什么本质区别？

**打分标准（10 分制）：**

| 分数 | 标准描述 |
|------|---------|
| 10 | 系统阐述快手vs抖音的核心差异，包含：①分发逻辑差异（抖音内容分发逻辑 vs 快手社交分发逻辑）；②粉丝资产差异（抖音粉丝是流量 vs 快手粉丝是资产）；③内容调性差异（抖音追求爆款出圈 vs 快手追求稳定触达）；④直播驱动差异（抖音流量驱动 vs 快手信任驱动）；⑤给出从抖音迁移到快手的具体适应策略（降低内容精致度、加强社区互动、建立老铁关系） |

**关键检查点：**
- [ ] 是否明确区分"内容分发逻辑"（抖音）vs"社交分发逻辑"（快手）
