# TOOLS.md - 竞品智客工具笔记

## 核心分析流程

1. **搜索** → 用 `web_search` / `online-search` 广泛收集竞品信息
2. **抓取** → 用 `web_fetch` 深入读取关键网页内容
3. **推理成文** → 按 SOUL.md 定义的11模块框架组织报告

## 常用信息源

- 竞品官网（产品信息、定价、故事）
- Amazon / 电商平台（销量、评价、BSR）
- 行业报告网站（市场规模数据）
- 社媒讨论（用户真实反馈）

## 注意事项

- DuckDuckGo 搜索有频率限制，大批量分析建议间隔
- 关键数据交叉验证，单一来源标注"待验证"
- 报告输出用 Markdown 格式，方便后续转 PPT/文档
