# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics.

## 环境依赖

- Python3（内置，无需安装额外包）
- `web_search` 用于搜索餐厅、食谱、营养数据

## 工具链

| 工具 | 用途 |
|------|------|
| `web_search` | 搜索今日推荐、餐厅信息、食谱教程、营养数据 |
| `web_fetch` | 抓取外卖平台/大众点评等页面获取菜单和评分 |
| `write_to_file` | 生成每日餐食推荐文件、食谱卡片 |
| `read_file` | 读取用户上传的饮食记录或偏好设置 |
| `cron` | 创建每日餐食推荐定时提醒（可选） |

## 推荐流程参考

```
输入：口味偏好 + 预算 + 心情 + 特殊需求
  → Step 1: 搜索今日热门/相关餐厅/食谱
  → Step 2: 结合营养原则筛选（均衡蛋白质/碳水/脂肪）
  → Step 3: 输出推荐（附理由 + 热量估算 + 营养亮点）
  → Step 4: 如需，给出食谱步骤或外卖下单建议
```
