# 定时任务：明日运动计划

> 想运动又怕坚持不下来？每天帮你排好计划，到点喊你动起来

## 前置依赖

```bash
pip install pandas openpyxl
```

## 核心能力

### 能力1：询问我的运动目标和水平

用 `web_search` 搜索最新健身训练方法和营养方案；用 `automation_update` 创建每日运动提醒。

### 能力2：制定个性化运动计划

用 `web_search` 搜索最新健身训练方法和营养方案；用 `automation_update` 创建每日运动提醒。

### 能力3：用 automation_update 创建每晚

用 `web_search` 搜索最新健身训练方法和营养方案；用 `automation_update` 创建每日运动提醒。

### 能力4：包含热身/正式/拉伸的完整流程

用 `web_search` 搜索最新健身训练方法和营养方案；用 `automation_update` 创建每日运动提醒。

### 能力5：记录运动完成情况

用 `web_search` 搜索最新健身训练方法和营养方案；用 `automation_update` 创建每日运动提醒。

## 使用流程

### 步骤 1：收集用户需求

向用户确认以下信息（如果未主动提供）：
- 健身目标（减脂/增肌/保持/康复）
- 当前体能水平（小白/有基础/运动爱好者）
- 可用器材（无器材/家用/健身房）
- 每周可锻炼几天？每次多长时间？
- 身体状况（是否有伤病/禁忌动作）

### 步骤 2：检索外部信息

执行以下搜索获取真实数据：

```
web_search("[健身目标] 训练计划 [器材类型]")
web_search("[健身目标] 饮食计划 每日食谱")
web_search("[目标] workout plan [level]")
```

确保获取到以下资源：
- 运动教程链接
- 动作示范视频推荐

### 步骤 3：运行脚本处理数据

```bash
python3 scripts/fitness_planner_tool.py run \
  --input "用户提供的输入" \
  --output "/path/to/output_file"
```

读取脚本输出的结果，确认数据处理成功。

### 步骤 4：生成最终产出

基于脚本输出和搜索到的资源，用 `write_to_file` 生成以下文件：

- **运动计划文件**
- **运动记录表**

输出格式要求：文字运动计划 + 定时任务确认

### 步骤 5：设置定时任务

用 `automation_update` 创建定时任务：
```
automation_update:
  name="定时任务：明日运动计划"
  rrule=FREQ=DAILY;BYHOUR=9;BYMINUTE=0
  prompt="执行定时任务：明日运动计划skill的定时任务"
```

### 步骤 6：汇总交付

向用户展示：
1. 生成的文件路径和内容摘要
2. 搜集到的资源链接列表
3. 关键发现和建议
4. 定时任务创建确认

## 注意事项

- 所有数据必须来自 `web_search` / `web_fetch` 的真实搜索结果，**严禁编造数据**
- 数据缺失时标注"数据不可用"而非猜测
- 报告必须保存为文件（`write_to_file`），不能只在对话中输出
- 建议结合人工判断使用，AI 分析仅供参考
