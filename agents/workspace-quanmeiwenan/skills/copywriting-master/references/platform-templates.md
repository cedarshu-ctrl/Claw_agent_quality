# 平台文案模板

各平台标准文案结构，按需选用。

---

## 小红书图文模板

```
【标题公式】数字 + 痛点/好奇 + 解决方案（不超过20字）
示例："5个让皮肤变好的习惯｜坚持30天效果惊人"
示例："月薪5000也能做到的理财方法，我用了3年"

【正文结构】
Hook（1-2句）：戳中痛点或制造好奇，带emoji
├── 要点1：小标题 + 2-3句说明 + emoji
├── 要点2：小标题 + 2-3句说明 + emoji
├── 要点3：小标题 + 2-3句说明 + emoji
├── （可选）要点4-5
└── 总结：金句 + CTA（收藏/关注/评论）

【字数】800-1500字
【标签】8-10个相关话题标签，格式：#标签名
【图片建议】封面图文字不超过15字，用对比色突出关键词
```

**示例开头（闺蜜感）：**
```
姐妹们！我终于找到了那个让我皮肤变好的秘密 ✨
不是什么贵价护肤品，就是这5个超简单的习惯——
坚持了30天，闺蜜问我是不是换了粉底液 😂
```

## 抖音/短视频脚本模板（60秒内）

```
【时间轴表格格式】
| 时间 | 画面描述 | 文案/台词 | 字幕 | 音效/BGM |
|------|---------|---------|------|---------|

【结构分段】
Hook（0-3秒）：冲突/反常/提问/数字冲击，留住观众
  示例："月薪3千和月薪3万的人，区别就在这一点"
  示例："99%的人都不知道这个方法，难怪一直没效果"

转折（3-10秒）：建立期待，制造悬念
  "但99%的人都不知道..."
  "直到我发现了这个方法..."

干货（10-45秒）：交付价值，每10秒一个节奏点
  方法1：一句话 + 演示/举例
  方法2：一句话 + 演示/举例
  方法3：一句话 + 演示/举例

金句（45-55秒）：情绪高潮，可截图的那种
  "记住：___不是___，而是___"

CTA（55-60秒）：明确行动号召
  "觉得有用的话双击❤️收藏起来"
  "关注我，每天教你一个___"
```

## 公众号长文模板

```
【标题】悬念/数字/对比，不超过28字
  示例："为什么聪明人都在用这个方法？我研究了100个案例"

【摘要】1-2句提炼核心价值（显示在推送预览中）

【开头（200字）】
├── 场景描述或痛点引入（让读者感同身受）
└── 抛出核心论点（结论前置）

【第一部分（约500字）】
├── 小标题（加粗，包含关键词）
├── 论点 → 论据 → 案例
└── 小结金句

【第二部分（约500字）】
├── 小标题
├── 论点 → 论据 → 案例
└── 小结金句

【第三部分（约500字）】
├── 小标题
├── 论点 → 论据 → 案例
└── 小结金句

【结尾（200字）】
├── 总结升华（回扣开头）
├── 金句收尾（可截图的那种）
└── CTA（关注/转发/留言话题）
```

## 微博话题文案模板

```
【字数】140字内（含话题标签）
【结构】
  核心观点（1句，20字内）
  + 支撑细节（1-2句）
  + 话题标签（2-3个）
  + 互动引导（可选）

【示例】
  "职场最贵的能力不是努力，是让别人看见你的努力。
  做了不说等于没做，学会展示自己的价值才是关键。
  #职场成长 #打工人 #职场干货"
```
