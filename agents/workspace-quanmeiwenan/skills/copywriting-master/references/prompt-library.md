## 小红书文案生成

```
你是一位小红书爆款文案写手，请按以下要求创作：

【品牌人设】{从identity.yaml加载，或描述品牌语气}
【选题】{选题标题}
【目标受众】{受众画像，如：25-35岁职场女性}
【目标关键词】{关键词1, 关键词2, 关键词3}
【参考爆款结构】{爆款标题或结构描述，可选}

输出要求：
1. 标题：3个备选，使用数字+痛点公式，不超过20字
2. 正文：800-1200字，段落清晰，每段1-2个emoji
3. 话题标签：8-10个相关话题
4. 关键词密度：主关键词出现3-5次
5. 开头第一句结论前置，禁止"众所周知"等油腻开头
```

## 抖音/短视频脚本生成

```
你是一位短视频脚本策划大师，请为以下选题撰写60秒脚本：

【选题】{选题标题}
【目标受众】{受众画像}
【核心卖点/信息】{1-3个核心信息点}
【风格】{搞笑/干货/情感/测评，选一}

输出格式（时间轴表格）：
| 时间 | 画面描述 | 文案/台词 | 字幕 | 音效/BGM |

确保：
- 前3秒有强钩子（冲突/反常/数字冲击）
- 每10秒一个节奏点，信息密度适中
- 45-55秒有可截图的金句
- 结尾有明确CTA（双击/收藏/关注）
```

## 公众号长文生成

```
你是一位公众号深度内容创作者，请按以下要求创作：

【选题】{选题标题}
【目标受众】{受众画像}
【核心论点】{文章要证明或传达的核心观点}
【关键词】{SEO目标关键词1, 关键词2}

输出要求：
1. 标题：3个备选，悬念/数字/对比风格，不超过28字
2. 摘要：1-2句，用于推送预览
3. 正文：1500-2500字，总分总结构
   - 开头200字：场景引入+核心论点
   - 3个主体部分，各约500字，每部分有小标题+金句
   - 结尾200字：总结升华+CTA
4. 每部分结尾必须有一句可截图的金句
5. 语气：专业但不失温度，避免学术腔
```

## 文案改写/风格迁移

```
请将以下文案改写为{目标风格}风格：

【原文案】
{粘贴原文案}

【目标风格】{闺蜜感/专业感/口语化/幽默感}
【目标平台】{小红书/抖音/公众号/微博}
【品牌人设】{语气要求，如：温暖、直接、不用"小编"}

改写要求：
- 保留核心信息和卖点，不改变事实
- 语气完全符合目标风格
- 句长适配目标平台（小红书≤20字/句，抖音≤15字/句）
- 输出改写版本 + 改动说明（改了哪些地方，为什么）
```

## 竞品爆款拆解

```
请分析以下爆款内容的结构，提炼可复用框架：

【爆款内容】
{粘贴爆款标题+正文，或描述爆款内容}

【平台】{小红书/抖音/公众号}

分析维度：
1. 标题公式：用了什么公式？为什么有吸引力？
2. 开头钩子：如何在前3句留住读者？
3. 内容结构：几个部分？每部分的逻辑关系？
4. 金句设计：哪些句子最容易被截图/转发？
5. CTA设计：如何引导互动？
6. 可复用框架：提炼成可套用的模板

输出：结构分析报告 + 可复用模板
```

## 人设一致性检查

```
请检查以下文案是否符合品牌人设：

【文案】
{粘贴待检查文案}

【品牌人设配置】
{粘贴identity.yaml内容，或描述品牌语气/禁用词/风格要求}

【目标平台】{平台名称}

检查项：
1. 是否使用了禁用词？（列出所有禁用词出现位置）
2. 语气是否符合品牌调性？（warmth/formality/energy）
3. 句长是否符合平台要求？
4. 是否有"小编"、"众所周知"等违规表达？
5. 整体人设得分（0-100）

输出：问题清单 + 修改建议 + 修改后版本
```
