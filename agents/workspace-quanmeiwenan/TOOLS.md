## ✍️ 何时使用「03-文案大师-Skill.md」

### 触发关键词（满足任一即触发）

| 任务类型 | 用户可能说的话 | Skill中的模板 |
|---------|--------------|--------------|
| **小红书文案** | "写一篇小红书"、"种草文"、"图文笔记" | 4.1 小红书图文模板 |
| **抖音脚本** | "写个抖音脚本"、"短视频脚本"、"拍个视频" | 4.2 短视频脚本模板 |
| **公众号长文** | "写篇公众号"、"深度文"、"行业文章" | 4.3 公众号长文模板 |
| **SEO优化** | "帮我优化下关键词"、"SEO一下" | 第五章 SEO优化模块 |
| **文案改写** | "改成xxx风格"、"换种说法"、"润色" | 第六章 Prompt库 |
| **竞品分析** | "分析下这个爆款"、"拆解下结构" | 第一章 工作流程 |
| **人设检查** | "检查下文案风格"、"符合品牌调性吗" | 3.2 人设一致性检查器 |

### Skill调用流程

```
1. 读取「03-文案大师-Skill.md」内容
2. 根据任务类型选择对应章节模板
3. 加载品牌人设（IDENTITY配置）
4. 按模板生成文案
5. 执行SEO检查
6. 适配目标平台格式输出
```

### 各平台输出格式

**小红书：**
- 标题：3个备选（数字+痛点）
- 正文：800-1500字，emoji适度
- 标签：8-10个话题标签

**抖音脚本：**
- 表格格式：时间 | 画面 | 文案 | 字幕 | 音效
- 60秒内完整结构：Hook→转折→干货→金句→CTA

**公众号：**
- 总分总结构
- 每部分500字左右
- 金句+CTA结尾

## 🤖 何时使用「agent-workspace-creator」

### 触发场景

| 场景 | 用户可能说的话 |
|-----|--------------|
| 创建新Agent | "帮我建一个agent"、"创建新workspace" |
| 从skill创建 | "根据这个skill生成agent"、"用xxx skill打造agent" |
| 从描述创建 | "我想要一个xxx助手"、"帮我设计一个agent" |
| 从聊天记录创建 | "从这段对话生成agent"、"把聊天记录变成agent" |

### 使用方式

不需要手动调用。当用户触发上述场景时，系统会自动：
1. 读取skill的SKILL.md
2. 构建Agent人设
3. 生成workspace文件
4. 注册新Agent

## 🔑 快速参考

**生成小红书文案的标准Prompt：**
```
请加载品牌人设配置。为选题"[选题标题]"生成小红书文案。
目标关键词：[词1, 词2, 词3]
参考爆款结构：[爆款链接或结构描述]
输出3个标题备选+完整正文+8-10个标签
```

**生成抖音脚本的标准Prompt：**
```
请为选题"[选题标题]"撰写60秒抖音脚本。
目标受众：[受众画像]
核心卖点：[1-3个核心信息]
按时间轴表格输出：时间|画面|文案|字幕|音效
```

**创建新Agent的标准Prompt：**
```
用[skill名称]skill帮我创建一个新agent
```
或者直接描述想要的agent人设，系统会自动调用agent-workspace-creator。
