## 工具链

| 工具 | 用途 |
|------|------|
| `web_search` | 查询目的地天气、交通、景点、住宿、美食 |
| `web_fetch` | 抓取旅游攻略网站的详细内容 |
| `write_to_file` | 生成完整行程方案文档（Markdown） |
| `read_file` | 读取历史出行记录 |
| `exec` (python3) | 运行 travel_planner_tool.py 处理数据 |

## 脚本用法

```bash
python3 skills/travel-planner/scripts/travel_planner_tool.py run \

```

## 推荐目的地分类

| 类型 | 适合场景 | 典型目的地（举例）|
|------|---------|----------------|
| 🌊 海边 | 夏季、亲子、情侣 | 巽寮湾/大鹏半岛/珠海 |
| 🌲 山林 | 春秋、徒步、自然爱好者 | 梧桐山/从化/莫干山 |
| 🎨 文艺 | 拍照打卡、闺蜜、情侣 | 大理/厦门/景德镇 |
| 🏙️ 城市周边 | 周末当天往返、朋友聚会 | 广州深圳周边 |
| ♨️ 温泉 | 冬季、家庭、放松 | 从化温泉/清远温泉 |

## 推荐车程范围

- 当天往返：1小时内
- 周末过夜：2小时内
- 超过2小时：不推荐（除非是小长假）

## 天气适配规则

| 天气 | 影响 | 处理方式 |
|------|------|---------|
| 晴/多云 | 无影响 | 正常推荐 |
| 雨天 | 户外景点受限 | 推荐室内/有雨备选的目的地 |
| 高温(>35°C) | 避免户外暴晒 | 推荐有遮蔽/室内为主 |
| 低温(<5°C) | 户外体感差 | 推荐温泉/室内/保暖景点 |

## 行程文档结构

每份行程文档应包含：
1. 目的地概览（车程/特色/适合人群）
2. 天气预报（出发日+行程日）
3. 时间表（详细到小时）
4. 交通方案（去程/返程）
5. 餐饮推荐（早/中/晚）
6. 景点详情（门票/预约/游览时长）
7. 雨天备用方案
8. 费用估算
9. 资源链接
