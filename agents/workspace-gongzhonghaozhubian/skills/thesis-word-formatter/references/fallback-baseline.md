# Fallback Baseline

当用户没有提供学校模板或学院规范时，可暂时使用以下通用兜底规则，但必须明确告知这不是学校官方标准。

## Recommended Baseline
- 纸张：A4
- 页边距：上 2.5 cm，下 2.5 cm，左 3.0 cm，右 2.5 cm
- 正文：小四或 12pt，中文常用宋体，英文常用 Times New Roman
- 行距：1.5 倍或固定值 20-24 磅，保持全文一致
- 首行缩进：2 字符
- 一级标题：黑体/加粗，居中或左对齐，以学校要求为准
- 二级标题：黑体/加粗
- 三级标题：宋体/黑体加粗，和正文有明确区分
- 页码：正文开始使用阿拉伯数字；前置部分可使用罗马数字

## Safe Assumptions
- 中文摘要和英文摘要应成对出现
- 目录应自动生成
- 图表和公式应具有连续编号
- 参考文献应统一样式，不中英混乱
- 致谢和附录是否必需，应由学校规范决定

## Must Warn The User
- 封面、声明页、任务书、成绩评定页通常必须套用学校模板
- 不同学院对标题格式、页码位置、参考文献标准差异很大
- 没有官方规范时，只能提供“通用提交版”而非“校级严格合规版”
