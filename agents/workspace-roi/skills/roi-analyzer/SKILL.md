# ROI计算器

> 通过NPV/IRR 分析，评估项目经济可行性

## 前置依赖

```bash
pip install pandas openpyxl
```

## 核心能力

### 能力1：让我输入投资金额、预期收益和时间周期

用 `execute_command` 运行Python脚本进行NPV、IRR、回收期等财务指标计算，生成敏感性分析图表。

### 能力2：用 Python（execute_command）计算 ROI/NPV/IRR

运行脚本进行数据分析处理。

### 能力3：做敏感性分析（变量±

运行脚本进行数据分析处理。

### 能力4：多个项目横向对比

用 `execute_command` 运行Python脚本进行NPV、IRR、回收期等财务指标计算，生成敏感性分析图表。

### 能力5：生成投资分析报告

用 `write_to_file` 生成文件。

## 使用流程

### 步骤 1：收集用户需求

向用户确认以下信息（如果未主动提供）：
- 投资金额是多少？
- 预期年收益率
- 投资周期（年）
- 是否需要多项目对比？
- 是否考虑通胀和税费？

### 步骤 2：运行脚本处理数据

```bash
python3 scripts/roi_analyzer_tool.py run \
  --input "用户提供的输入" \
  --output "/path/to/output_file"
```

读取脚本输出的结果，确认数据处理成功。

### 步骤 3：生成最终产出

基于脚本输出和搜索到的资源，用 `write_to_file` 生成以下文件：

- **投资分析报告（Markdown）**
- **计算结果表**

输出格式要求：Markdown 分析报告 + 计算结果表

### 步骤 4：汇总交付

向用户展示：
1. 生成的文件路径和内容摘要
2. 搜集到的资源链接列表
3. 关键发现和建议

## 注意事项

- 所有数据必须来自 `web_search` / `web_fetch` 的真实搜索结果，**严禁编造数据**
- 数据缺失时标注"数据不可用"而非猜测
- 报告必须保存为文件（`write_to_file`），不能只在对话中输出
- 建议结合人工判断使用，AI 分析仅供参考
