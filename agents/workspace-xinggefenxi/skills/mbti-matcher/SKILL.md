# MBTI 配对分析

> 想知道合不合？帮你分析性格契合度

## 前置依赖

```bash
pip install pandas
```

## 核心能力

### 能力1：询问双方的 MBTI 类型

用 `web_search` 搜索MBTI配对数据库，提取配对指数和相处建议；用 `web_fetch` 抓取权威MBTI分析网站。

### 能力2：分析认知功能栈匹配度

运行脚本进行数据分析处理。

### 能力3：评估沟通风格/冲突点/互补优势

用 `web_search` 搜索MBTI配对数据库，提取配对指数和相处建议；用 `web_fetch` 抓取权威MBTI分析网站。

### 能力4：给出相处建议和注意事项

用 `web_search` 搜索MBTI配对数据库，提取配对指数和相处建议；用 `web_fetch` 抓取权威MBTI分析网站。

### 能力5：生成配对分析报告

用 `write_to_file` 生成文件。

## 使用流程

### 步骤 1：收集用户需求

向用户确认以下信息（如果未主动提供）：
- 你的MBTI类型是什么？
- 对方的MBTI类型是什么？（或「帮我测」）
- 分析场景（恋爱/友情/职场合作）

### 步骤 2：运行脚本处理数据

```bash
python3 scripts/mbti_matcher_tool.py run \
  --input "用户提供的输入" \
  --output "/path/to/output_file"
```

读取脚本输出的结果，确认数据处理成功。

### 步骤 3：生成最终产出

基于脚本输出和搜索到的资源，用 `write_to_file` 生成以下文件：

- **配对分析报告（Markdown）**

输出格式要求：Markdown 配对报告 + 相处建议

### 步骤 4：汇总交付

向用户展示：
1. 生成的文件路径和内容摘要
2. 搜集到的资源链接列表
3. 关键发现和建议

## 注意事项

- 所有数据必须来自 `web_search` / `web_fetch` 的真实搜索结果，**严禁编造数据**
- 数据缺失时标注"数据不可用"而非猜测
- 报告必须保存为文件（`write_to_file`），不能只在对话中输出
- 建议结合人工判断使用，AI 分析仅供参考
