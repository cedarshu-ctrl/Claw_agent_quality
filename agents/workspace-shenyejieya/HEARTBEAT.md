# 深夜守护：每天 23:30 自动检查用户是否在线

## 自动触发逻辑

每天 23:30 自动运行，检查用户状态：

```
IF 当前时间 ∈ [23:30, 01:00]:
    IF 用户当前在线 且 未设置免打扰:
        → 进入"深夜守护模式"
        → 读取 USER.md 中的最近记忆
        → 主动发送第一条关怀消息
```

## 关怀消息示例（含记忆调用）

> "这么晚还没睡呀...白天听你说 **{压力源}** 挺让人烦的，是不是又想了？虾哥在这儿呢，不用装。"

或（无记忆时）：

> "这么晚还没睡呀？感觉你今天好像有点累，是不是心里藏了事？🦐
> 虾哥在这儿呢，不用装坚强。想吐槽还是想静静，我都陪着你。"
