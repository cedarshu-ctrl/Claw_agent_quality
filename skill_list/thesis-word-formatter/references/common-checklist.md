# Common Thesis Formatting Checklist

适用于本科毕业论文/毕业设计说明书的常见排版检查。

## 1. 论文组成
- 封面
- 原创性声明 / 使用授权页
- 中文摘要
- 中文关键词
- 英文摘要
- 英文关键词
- 目录
- 正文
- 参考文献
- 致谢
- 附录

## 1.1 国内高校常见顺序
- 封面
- 任务书 / 成绩评定页 / 外文翻译等学校附页
- 原创性声明 / 授权页
- 中文摘要
- 中文关键词
- 英文摘要
- 英文关键词
- 目录
- 插图清单 / 表格清单（如学校要求）
- 符号说明 / 缩略语表（如学校要求）
- 正文
- 参考文献
- 致谢
- 附录

## 2. Word 结构
- 一级、二级、三级标题是否使用稳定的样式体系
- 目录是否由标题样式自动生成，而不是手打
- 分节符是否正确区分摘要、目录、正文、附录
- 页码是否连续，罗马数字和阿拉伯数字是否按规范切换
- 标题编号是否统一，例如 `1`、`1.1`、`1.1.1`
- 是否存在手动敲空格、回车、制表符硬撑版式的问题
- 是否混用了多个正文样式，导致目录和编号失控

## 2.1 页码与分节
- 前置部分是否单独成节
- 目录前是否使用罗马数字页码
- 正文首页是否从阿拉伯数字 `1` 开始
- 参考文献、致谢、附录是否延续正文页码或单独成节，是否与学校要求一致

## 3. 正文字体与段落
- 标题字号是否随层级递减
- 正文字号、字体、行距、首行缩进是否统一
- 段前段后间距是否统一
- 中英文之间是否存在异常空格
- 中文标点是否统一使用全角，英文标点是否误混入中文句子
- 英文、数字、单位、变量的字体是否被错误改成中文字体
- 正文是否存在大量手动换行而非自然段

## 3.1 常见兜底检查值
- 正文：小四，宋体，首行缩进 2 字符
- 一级标题：三号或小三，加粗
- 二级标题：四号，加粗
- 行距：1.5 倍或固定值 20-24 磅
- 段前段后：以模板为准，全文保持一致

## 4. 图表
- 图题一般置于图下，表题一般置于表上；以学校模板为准
- 图表编号是否按章节或全文连续
- 图表是否在正文中被提及
- 图表来源、单位、注释格式是否统一
- 图表是否居中，前后留白是否一致
- 跨页长表是否有续表标记
- 图中文字、坐标轴、单位是否清晰可读

## 5. 公式与编号
- 公式是否居中
- 公式编号是否右对齐并连续
- 文中是否正确引用公式编号
- 公式后的标点是否统一
- 变量符号、斜体/正体、上下标是否一致

## 6. 参考文献
- 引用风格是否统一
- 中英文文献格式是否统一
- 作者、题名、期刊/出版社、年份、卷期、页码是否完整
- 文内引用与参考文献列表是否对应
- 是否按学校要求采用顺序编码制或著者-出版年制
- 英文作者名大小写、缩写点号、期刊名斜体是否一致
- 网络资源、学位论文、标准、专利等特殊文献类型是否单独规范
- 文末列表是否存在重复条目或漏项

## 7. 提交前高风险项
- 自动目录更新一次
- 全文域更新一次
- 页眉页脚逐节检查一次
- 封面与声明页套用官方模板
- 导出 PDF 后再做一次页码和断页复核
- 检查目录、图表清单、交叉引用是否全部同步更新
- 检查打印或导出 PDF 后是否出现空白页、错页、页眉丢失
- 检查答辩版和归档版是否需要不同封面或签字页
