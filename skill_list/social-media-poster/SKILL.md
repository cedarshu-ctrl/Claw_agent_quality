# 社媒发布 (Social Media Poster)

## 功能

- 多平台支持：微信、微博、抖音、小红书、LinkedIn、Twitter
- 内容一键发布：一次编辑，多平台同步
- 定时推送：设置发布时间，自动执行
- 效果追踪：阅读量、点赞、评论数据追踪
- 素材管理：图片、视频素材库

## 使用场景

1. 自媒体运营：一人管理多个账号
2. 品牌营销：统一发布营销内容
3. 电商推广：商品上架同步推广

## 配置

需要配置各平台 API credentials：
- 微信公众号 API
- 微博 Open API
- 抖音开放平台
- 小红书商业化 API
